
# HoloTalk

HoloTalk is a modern, real-time messaging application inspired by Discord, built for seamless comunication between individuals and groups. It combines instant messsaging, voice channels and community-driven features into one uniied platform.
## 🙏 Acknowledgements

- [Gin Gonic](https://github.com/gin-gonic/gin) - Fast HTTP web framework written in Go.
- [WebSocket Protocol](https://developer.mozilla.org/en-US/docs/Web/API/WebSockets_API) - For real-time communication.
- [Discord](https://discord.com/) - Main inspiration for UI/UX and feature ideas.
- [Golang UUID](https://github.com/google/uuid) - For generating unique identifiers.
- [Docker](https://www.docker.com/) - For containerizing the application.
- [Visual Studio Code](https://code.visualstudio.com/) - Preferred IDE for development.
